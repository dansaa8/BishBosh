Bish-Bosh 2.0

Information about this app:

Iterations input:

Counts how many times a number from 1-(the entered value) is supposed to loop for

Bish input:

Every time the entered number in Bish field is divisble by any number from 1 to the entered iterations input; Bish will be printed

Bosh input:

Every time the entered number in Bosh field is divisble by any number from 1 to the entered iterations input; Bosh will be printed

Other cases:

If the number is divisble by both Bish and Bosh, then Bish-Bosh will be printed

If the number is not divisable by either Bish or Bosh, then the number will be printed instead
